# sorting-algo-func.f90

🦄🦄🦄 Fortranで整列アルゴリズムを実装するための関数群です。  

## 実行方法

以下のコマンドでソースコードをコンパイルし、実行ファイルを生成します。  

```shell
make
```

`./bin/make`ファイルが生成されます。  

```shell
./bin/make
```

---

コンパイルされたファイルを削除する場合は以下のコマンドを実行します。  

```shell
make clean
```
